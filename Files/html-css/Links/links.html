HyperLinks

Meu github

O parâmetro target="_blank" faz o hyperlink abrir em uma nova janela

O parâmetro rel="" indica funcionalidades do link:
next: próxima parte do documento atual
prev: parte anterior do documentoa atual
author: indica link para autor do artigo atual
external: indica que é link externo

O parâmetro download="" recebe um arquivo que será baixado ao clicar
Deverá seguir o parâmetro type="". Tipos aceitos:

  • aplication/zip
  • application/pdf
  • text/html
  • text/css
  • text/javascript
  • video/mp4
  • video/H264
  • video/JPEG
  • audio/mpeg
  • audio/aac
  • font/ttf
  • image/jpeg
  • image/png
